# DeepAge 

Have you found an old image for which you can't tell when it was taken? Worry not, DeepAge does that for you. *If the image happens to be from somewhere between 1900 and 1965*.

Python 2.7.

Dependencies: 
* Keras 1.2.2
* OpenCV 3.1.0
* Numpy 1.12.1
* BeautifulSoup 4.5.3
* Requests 2.13.0
* Scikit-learn 0.18.1
* Matplotlib 2.0.1
